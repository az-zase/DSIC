# Часть A — Параметр отлипания μ_d

**Статус величины:** рабочий параметр, зафиксированный по одному наблюдению (CMB). **Не** выведен из ядра. **Не** имеет пока второго независимого подтверждения.

---

## A.1 Одной фразой

`μ_d` — фаза, на которой ранняя «каша» позволяет своей материи отделяться, и модель переключается с режима ранней Вселенной на режим поздней. Численно: **`μ_d = 0.980638`**, что отвечает красному смещению **`z ≈ 4.53`**.

---

## A.2 Зачем она понадобилась

Без поправки DSIC недомеряет расстояние до реликтового излучения: даёт ≈ 10936 Мпк против измеренных Planck ≈ 13873 Мпк — недобор ~ 21 %. Причина: в круговой геометрии DSIC горизонт частиц конечен «бесплатно», и почти весь путь набирается уже к `z ≈ 10`, а дальше интеграл расстояния выдыхается.

При этом ближняя и средняя Вселенная (`z < 2.3`) считается верно — на данных DESI DR2 выходит `χ²/dof ≈ 0.78`. Значит воздействовать надо **только** на дальний конец, не трогая ближний. Параметр `μ_d` ровно это и делает: включает поправку только глубоко в прошлом и жёстко отсекает её на пороге отлипания.

---

## A.3 Физическая картина (как DSIC это видит)

Ранняя эпоха — почти монолитная объектность, в которой пространства практически нет. Теснота объектности создаёт невыносимое давление. По мере того как пространство прибывает, давление спадает.

**Аналогия пресса.** Набор предметов сжали. Когда пресс отпускает, предметы сначала ещё распирают (демпфируют), а в какой-то момент **отлипают от стенок** — и после этого давления на стенки больше нет, хотя пресс продолжает расходиться. Точка отлипания и есть `μ_d`.

**Следствие для света.** Пока объекты «приклеены» (фаза ближе к каше, чем `μ_d`), плотность материи удлиняет путь света — добирается недостающее расстояние. После отлипания (фаза дальше от каши) поправки нет вовсе, и работает обычная поздняя геометрия DSIC.

---

## A.4 Как считается

Обозначения — наблюдаемая фаза `μ ∈ [0,1]` (доля объектности), масштаб пространства `P = √(1−μ²)`, масштабный фактор `b(μ) = √(1−μ²)/μ²`. Параметры: `μ₀ = 0.76` (фаза «сегодня»), `H₀ = 67.4`.

**Шаг 1. Базовое расстояние DSIC (без поправки):**
```
D_M(μ_e) = (c/k)·[√(1−μ₀²)/μ₀²]·[√(1−μ₀²) − √(1−μ_e²)]
c/k      = (c/H₀)·(2−μ₀²)/(1−μ₀²) ≈ 14978 Мпк
```

**Шаг 2. Фаза реликта.** Из `1+z = b(μ₀)/b(μ_e)` при `z = 1090` → `μ_cmb ≈ 0.99999947`.

**Шаг 3. Плотность как добавка к пути.** Давление берётся в естественной форме — обратной масштабу пространства `1/P`, отсеченное порогом отлипания:
```
поправка(μ) = 1/√(1−μ²)   при   μ > μ_d
поправка(μ) = 0            при   μ ≤ μ_d
```
Давление бесконечно велико у самой каши (`μ → 1`) — это физически верно (нулевое пространство = бесконечная теснота). Отсечка на `μ_d` не даёт сумме по пути «взорваться»: сочетание «бесконечность у каши + жёсткий обрыв на пороге» делает поправку конечной.

**Шаг 4. Определение `μ_d`.** Порог фиксируется требованием: при естественной амплитуде (множитель = 1) добавленное расстояние ровно закрывает недобор до Planck. (`∫ 1/√(1−μ²) dμ = arcsin μ`):
```
(c/k)·[ arcsin(μ_cmb) − arcsin(μ_d) ] = 13873 − 10936 ≈ 2937 Мпк
```
Решение: **`μ_d = 0.980638`**, т.е. **`z_detach ≈ 4.53`**.

**Шаг 5. Добор расстояния:**
```
D_M_испр(μ_e) = D_M(μ_e) + (c/k)·[ arcsin(μ_e) − arcsin(μ_d) ]   при   μ_e > μ_d
D_M_испр(μ_e) = D_M(μ_e)                                          при   μ_e ≤ μ_d
```
Для `μ_e` ниже порога (`z < 4.53`) добавка равна нулю — поздняя Вселенная не меняется.

---

## A.5 Что это затрагивает, а что нет

| проба | данные | результат |
|---|---|---|
| расстояние до CMB | Planck `d* = 13873` Мпк | совпадает (по построению `μ_d`) |
| BAO `D_M/D_H` | DESI DR2 (сырые) | `χ²/dof = 0.783` — без изменений (было 0.78) |

Поправка активна только выше `z ≈ 4.53`, поэтому все точки DESI (`z ≤ 2.33`) её не чувствуют. Ближний результат сохранён полностью.

---

## A.6 Как DSIC трактует реликтовый отпечаток

- Реликтовый отпечаток, это стоп-кадр формы объектности на поздней стадии каши, когда пространство уже начало проступать и свет впервые смог выйти из монолитной объектности: контраст на карте есть контраст плотности проступающего пространства.
- Каша в DSIC — не строго нулевое состояние, а **отступ от нуля** (`μ_cmb ≈ 0.99999947`): пространство уже чуть проступило, и появившийся контраст даёт отпечаток.
- Знак контраста (следует из структуры модели): холоднее = свет вышел из области с большей объектностью в проступившее пространство. Значит **холодные/тёмные участки ↔ плотная объектность (монолит); горячие ↔ проступившие поры пространства.** Малость контраста (~ 10⁻⁵) согласуется с тем, что у порога поры пространства только зарождаются.
- **Важно:** реликт (`μ_cmb`) — это наблюдаемое «дно», а `μ_d` (`z ≈ 4.53`) — **порог отлипания**, более поздний край той же ранней зоны.

---

## A.7 Что значит этот параметр — и почему μ_d пока не "легален"

**Текущий статус `μ_d`:**
- Зафиксирована по **одному** наблюдению — расстоянию до CMB.
- Проверен на независимых данных (наклон остатков DESI на `z ≈ 0.9`) — **след не обнаружен**: порог `z ≈ 4.53` лежит выше всех точек DESI и до них не дотягивается.

**Что всё же приобретено:** убрав произвол амплитуды давления (взята естественная форма `1/P` без множителя), были сокращены две неизвестные ручки («где отлипает» и «насколько сильно давит») до одной — и порог определился сам, `z ≈ 4.53`. Это уже не голая подгонка значения, а «при естественной силе распора отлипание обязано быть здесь».

# Часть B — Гравитация и распределение пространства

> Феноменологический модуль, **не выведенный из ядра DSIC**. Сохранён как направление, а не как часть модели. Использует ньютоновские величины (`G, M_i`), которых в перетекании `S ↔ O` нет.

## B.1 Общая суть

Когда объекты схлопываются, между ними появляется пространство — но не поровну. Чем сильнее объекты связаны гравитацией, тем отчётливее выражен их общий центр падения, и тем меньше высвобождается пространства между ними; между слабо связанными гравитацией объектами — больше. Петля самоусиливается: разлёт ослабляет связь → область получает больше пространства → разлёт ускоряется. Плотные структуры остаются компактными. Пространство перераспределяется, глобальный баланс сохраняется, анизотропия локальна.

## B.2 Обозначения (локальные, не из ядра)

`r_ij` — соотношение масштабов пары; `M_i` — масса объекта; `G = 1`; `κ` — свободный параметр (темп перераспределения).

## B.3 Формулы

```
связанность:         g_ij = G·M_i·M_j / r_ij²
градиент потенциала: |∇Φ_ij| = | Σ_k G·M_k·(mid_ij − pos_k)/|mid_ij − pos_k|³ |
вес пространства:     w_ij = (1/|∇Φ_ij|) / Z,   Z = Σ_{i≠j} 1/|∇Φ_ij|
прирост:             Δr_ij = κ·|ΔP|·w_ij / (M_i + M_j)
эволюция:            r_ij(φ+Δφ) = r_ij(φ) + Δr_ij        (только растёт)
сохранение:          Σ_{i≠j} Δr_ij·(M_i + M_j)/κ = |ΔP(φ)|
```

Свойства весов: `w_ij ≥ 0`, `Σ w_ij = 1`. Пара в пустоте (плоское поле) получает большой вес; пара в кластере (крутое поле) — малый.

# Общие открытые вопросы

Обе части упираются в одно и то же: **в ядре DSIC нет второй независимой величины — связи между объектами.** В полюсах `S, O` есть только перетекание масштаба. А распор каши, и гравитационная связанность требуют этой второй сущности. Поэтому ни `μ_d`, ни гравитационный модуль из ядра не выводятся — их введение есть **расширение** модели, а не следствие.
