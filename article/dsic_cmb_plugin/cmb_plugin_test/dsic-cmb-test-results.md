# DSIC CMB-плагин — Результаты проверки

**Расширение:** Параметр отлипания `μ_d` (ранняя Вселенная, расстояние до реликта)
**Автор:** Азамат Засеев
**Скрипт:** `dsic-cmb-test.py` (самодостаточный, без внешних файлов)

> **Статус расширения.** `μ_d` — феноменологическая надстройка над ядром DSIC, **не выведенная из ядра**. Параметр зафиксирован по **одному** наблюдению (расстояние до поверхности последнего рассеяния, Planck) и не имеет второго независимого подтверждения. Данная проверка **не доказывает** `μ_d`; она подтверждает, что арифметика плагина воспроизводима, опорная цифра Planck верна, поздняя Вселенная поправкой не затронута, и честно показывает, что у параметра ноль степеней свободы.

---

## 2. Геометрия ядра до реликта

Параметры ядра DSIC на ранней шкале: `μ₀ = 0.76`, `H₀ = 67.4` (те же, что для BAO, хронометров и возраста в основном тесте). Перевод геометрии в мегапарсеки: `c/k = (c/H₀)·(2−μ₀²)/(1−μ₀²)`.

```
c/k             = 14978.2 Мпк
μ_cmb(z=1089.92) = 0.9999994681      (из 1+z = b(μ₀)/b(μ_e))
D_M база до CMB  = 10936.2 Мпк       (геометрия ядра, без поправки)
d_LSS Planck     = 13872.8 Мпк
недобор          = 2936.6 Мпк  (21.2 %)
```

Без поправки ядро **недомеряет** расстояние до реликта на ~21 %. Причина: в круговой геометрии DSIC горизонт частиц конечен, и почти весь путь набирается уже к `z ≈ 10`, а дальше интеграл расстояния выдыхается. Воздействовать нужно **только** на дальний конец, не трогая ближний.

---

## 3. Фиксация порога `μ_d`

Поправка берётся в естественной форме (обратной масштабу пространства, `1/√(1−μ²)`), отсечённой порогом отлипания; её интеграл — `arcsin μ`. Амплитуда взята естественной (`A = 1`, без свободного множителя). Порог определяется требованием замкнуть недобор:

```
(c/k)·[arcsin(μ_cmb) − arcsin(μ_d)] = 2936.6 Мпк

μ_d      = 0.980640
z_detach = 4.5259
D_M испр = 13872.8 Мпк   (= d_LSS Planck, замыкание точное)
```

---

## 4. Контроль: поздняя Вселенная не затронута

Поправка активна **только** выше порога `z_detach ≈ 4.53`. Ниже порога она строго равна нулю:

| `z` | `D_M` база, Мпк | поправка, Мпк |
|---|---|---|
| 0.50 | 1932.8 | +0.0 |
| 1.00 | 3384.2 | +0.0 |
| 2.33 | 5792.7 | +0.0 |
| 4.00 | 7335.6 | +0.0 |
| 4.53 | 7653.3 | +0.0 |

| 5.00 | 7896.9 | +220.4 ← включена |

Максимальная поправка при `z ≤ z_detach` равна `0.00` Мпк. Все 6 бинов DESI DR2 (`z ≤ 2.33`) поправки не чувствуют — `χ²/dof` на BAO не меняется, результаты основного теста сохранены полностью.

---

## 5. Степени свободы — честный разбор

Уравнение `A·(c/k)·[arcsin(μ_cmb) − arcsin(μ_d)] = недобор` имеет **кривую** решений `(A, μ_d)`: любая амплитуда даёт своё `μ_d`, и все они одинаково точно воспроизводят расстояние до CMB.

| `A` | `μ_d` | `z_detach` |
|---|---|---|
| 0.5 | 0.923706 | 1.506 |
| 0.8 | 0.969868 | 3.344 |
| **1.0** | **0.980640** | **4.526** ← естественная |
| 1.2 | 0.986514 | 5.691 |
| 2.0 | 0.995097 | 10.266 |
| 5.0 | 0.999190 | 26.922 |

Выбор `A = 1` («естественная форма `1/P` без множителя») фиксирует `μ_d`, но это **выбор формы**, а не вывод из данных. Одно наблюдение (`d_LSS`) фиксирует один параметр — **степеней свободы ноль**. Это **калибровка, а не предсказание**: значение `z_detach ≈ 4.53` нельзя предъявлять как результат, поскольку оно следует из выбора амплитуды.

## 6. Где `μ_d` оставляет проверяемый след

Поправка к `D_M` становится заметной только глубоко за порогом:

| `z` | объект | поправка к `D_M` |
|---|---|---|
| 4.53 | порог отлипания | +0.0 % |
| 5.00 | галактики JWST | +2.8 % |
| 6.00 | квазары | +7.1 % |
| 8.00 | реионизация | +12.4 % |
| 11.00 | первые галактики | +16.6 % |
| 1089.92 | CMB (калибровка) | +26.9 % |

**Досягаемость зондов:**

- DESI DR2 (есть): `z ≤ 2.33` — ниже порога;
- DESI DR3 (~2027): `z ≤ 3.55` — ниже порога;
- DESI-II / LBG (~2029): `2.5 < z < 3.5` — ниже порога.

> **Вывод о проверяемости.** Радиальный BAO в окне `2.33 < z < 4.53` расходится с ΛCDM (до ~4 % на `z ≈ 3.5`, в пределах досягаемости DR3), но это след **ядра** DSIC (`D_H = c/H` зависит только от `μ₀`), а **не** `μ_d`. Сам порог влияет лишь на `D_M` выше `z ≈ 4.5`, где точных стандартных линеек, кроме самой CMB-калибровки, нет. Второго **независимого** якоря для `μ_d` в досягаемых данных не существует — плагин остаётся некалибруемым извне до появления точной шкалы расстояний на `z ≳ 5–6`.

## 7. Итог

Что **проверено и сходится:**

- арифметика плагина воспроизводима (все числа совпадают с независимым пересчётом);
- опорная цифра Planck верна и внутренне согласована (`d_LSS = r*/θ* = 13872.8` Мпк);
- замыкание до расстояния до реликта точное (`D_M_испр = 13872.8` Мпк);
- поздняя Вселенная (`z ≤ 4.53`) поправкой **не затронута** — основной результат DSIC сохранён.

Что **заявлено честно и не подтверждено:**

- `μ_d` — калибровка по одному наблюдению, **ноль степеней свободы**; `z_detach ≈ 4.53` следует из выбора амплитуды `A = 1`, а не из данных;
- второго независимого теста `μ_d` в досягаемых данных нет.

Плагин корректно держится **отдельным файлом** со статусом феноменологической надстройки. Его нельзя ни подтвердить, ни опровергнуть текущими данными; это прямо указывается, а `z_detach ≈ 4.53` не подаётся как предсказание. Сильная проверяемая сторона остаётся за **ядром** — расхождение `D_H/r_d` на `z ≈ 3–3.5`, которое DESI DR3 реально проверит.
