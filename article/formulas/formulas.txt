𝒮[φ] = ∫ ½ φ̇² dt — действие свободного потока φ; при варьировании получается уравнение φ̈ = 0 (равномерное течение без ускорения)

φ = ω·t — фаза растёт равномерно со скоростью ω, задавая единственную динамику времени

S = R·sin(φ/2) — пространственный масштаб со знаком, меняет знак при прохождении нуля

O = R·cos(φ/2) — масштаб объектности со сдвигом по фазе относительно S

S² + O² = R² — сохранение нормы: геометрия окружности, перераспределение между S и O

W = S·O′ − O·S′ = −R²/2 — Вронскиан-константа, фиксирующая квадратурное соотношение компонент

σ = sign(S), sign(O) — знаковые фазы компонент

O_full = O · σ_S — объектность наследует знак пространства

π : O(φ) ↦ |O(φ)| — проекция, убирающая знак (переход к наблюдаемой величине)

P = |S| = R·√(1−μ²) — наблюдаемое пространство (неотрицательная проекция S)

Q = |O| = R·μ — наблюдаемая объектность (неотрицательная проекция O)

μ = |cos(φ/2)| = Q/R — наблюдаемая фаза, доля объектности в системе

P² + Q² = R² — сохранение нормы в наблюдаемой (модульной) проекции

L(μ) ∝ μ — масштаб линейки растёт пропорционально μ

dτ = dt/μ — ход времени наблюдателя замедляется как 1/μ

c = const — фиксированная скорость света как инвариант шкал

b(μ) = √(1−μ²) / μ² — масштабный фактор, определяющий наблюдаемую космологию

1 + z = b(μ₀) / b(μ_e) — красное смещение как отношение масштабов

|H_τ| = (2−μ²) / (1−μ²) — темп расширения (характерная космологическая скорость)

q = (−μ⁴+6μ²−4)/(μ⁴−4μ²+4) — параметр замедления расширения

r = (μ⁶−16μ⁴+20μ²−8)/(μ⁶−6μ⁴+12μ²−8) — statefinder-параметр r

s = (r−1) / [3(q−½)] — statefinder-параметр s

χ = (c/k)·[√(1−μ₀²)/μ₀²]·[√(1−μ₀²) − √(1−μ_e²)] — сопутствующее расстояние в замкнутой форме

d_L = (1+z)·χ — светимостное расстояние

DM = 5·log₁₀(d_L/10пк) — модуль расстояния

Om(z) = (E²−1)/((1+z)³−1) — диагностическая функция космологии

popravka ∝ 1/P = 1/√(1−μ²) — поправка к пути в связанном режиме среды

μ_d = 0.9806 (z_detach ≈ 4.53) — порог отлипания (константа перехода режима связанности)

D_M испр = D_M + (c/k)·[arcsin μ_cmb − arcsin μ_d] — исправленный модуль расстояния до реликтового излучения

δ″ + (2 + dlnH/dlna)·δ′ − 1½·Ω_m(a)·δ = 0 — уравнение роста линейных возмущений

fσ₈(z) = f·σ₈·δ(z)/δ(0) — наблюдаемая комбинация роста структуры
