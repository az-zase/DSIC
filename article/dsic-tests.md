# DSIC — Результаты эмпирической проверки

**Модель:** Dual Scale Inversion Cosmology (DSIC)
**Автор:** Азамат Засеев
**Сравнение:** DSIC против плоской ΛCDM на идентичных данных, при равном числе свободных параметров.

> **Краткий вывод.** На геометрии расширения поздней Вселенной (`z < 2.3`) DSIC с единственным параметром `μ₀` статистически неотличима от ΛCDM на сверхновых и слегка предпочтительнее на совместном фите SN+BAO. Однако `Om(z)`-диагностика на текущих данных модели **не разрешает** (знак Δχ² зависит от нормировки `H₀`), а на высоком `z` DSIC даёт чёткие фальсифицируемые предсказания, отличные от ΛCDM. Результаты воспроизводимы: каждый скрипт сам скачивает публичные каталоги и считает χ² с полной ковариацией.

---

|---|---|---|---|---|---|

## 2. Сверхновые: диаграмма расстояний

### 2.1 Космологический фит (Pantheon+, полная ковариация STAT+SYS)

Полный каталог Pantheon+ (1701 SN; в фит идут 1580 при `z > 0.01`, без калибраторов), полная ковариационная матрица STAT+SYS (1701×1701), минимизация χ² с полной ковариацией. Свободны два параметра у каждой модели.

```
DSIC : μ₀ = 0.7603,  H₀ = 73.04,  χ² = 1387.0,  χ²/dof = 0.8789
ΛCDM : Ωm = 0.3316,  H₀ = 73.25,  χ² = 1387.1,  χ²/dof = 0.8790

Δχ² (DSIC − ΛCDM) = −0.13
```

Геометрия одного параметра `μ₀` воспроизводит ту же зависимость расстояния от красного смещения, что и ΛCDM с параметром плотности `Ωm`, — без тёмной энергии. Разница качества подгонки на уровне статистического шума (`|Δχ²| ≪ 2`).

### 2.2 Якорный фит: калибраторы + `M` как явный параметр

Более строгая постановка: калибраторы (77 SN с цефеидными расстояниями) не выбрасываются, а используются как абсолютный якорь; абсолютная звёздная величина `M` фитится явно; `H₀` становится настоящим выводом из якоря, а не поглотителем сдвига. Три свободных параметра у каждой модели (`μ₀`/`Ωm`, `H₀`, `M`).

```
DSIC : μ₀ = 0.7600,  H₀ = 73.35,  M = −19.243,  χ²/dof = 0.8783
ΛCDM : Ωm = 0.3308,  H₀ = 73.53,  M = −19.244,  χ²/dof = 0.8783

Δχ² = ΔAIC = ΔBIC = +0.05
```

На полностью откалиброванных данных DSIC статистически **неотличима** от ΛCDM. `H₀ ≈ 73` выходит как вывод из якоря и наследует известное «напряжение Хаббла» относительно ранней Вселенной (`H₀ ≈ 67`) — одинаково у обеих моделей. То есть DSIC не создаёт нового напряжения и не устраняет существующего.

### 2.3 Устойчивость: STAT-ONLY против STAT+SYS

Контроль: тот же фит с матрицей только статистических ошибок (без систематик). Если параметры и форма почти не сдвигаются — результат не держится на систематиках.

| | `μ₀` / `Ωm` | `H₀` | `χ²/dof` |
|---|---|---|---|
| DSIC, STAT+SYS | 0.7603 | 73.04 | 0.8789 |
| ΛCDM, STAT+SYS | 0.3316 | 73.25 | 0.8790 |
| DSIC, STAT-ONLY | 0.7661 | 72.86 | 0.9316 |
| ΛCDM, STAT-ONLY | 0.3466 | 73.05 | 0.9319 |

```
Δχ² (DSIC − ΛCDM), STAT-ONLY = −0.39
```

`μ₀` сдвигается лишь с 0.7603 до 0.7661, форма зависимости сохраняется, DSIC остаётся вровень с ΛCDM. Рост `χ²/dof` при удалении систематик — ожидаемый эффект (систематики раздувают ошибки), одинаковый у обеих моделей.

## 3. Совместный фит SN + BAO

Сверхновые Pantheon+SH0ES (1657 SN, якорь) и BAO DESI DR2 (6 бинов, `0.5 < z < 2.33`) с **полной 12×12 ковариацией** `D_M`–`D_H` (учтена корреляция поперечного и радиального масштаба внутри бина). Масштаб звуковой дорожки `r_d` — свободный параметр (не сокращается через отношение). Четыре свободных параметра у каждой модели.

```
DSIC : μ₀ = 0.7615,  H₀ = 73.33,  M = −19.243,  r_d = 136.3 Мпк
       χ²(SN) = 1452.7,  χ²(BAO) = 7.96,  тотал = 1460.7
ΛCDM : Ωm = 0.3044,  H₀ = 73.74,  M = −19.247,  r_d = 136.9 Мпк
       χ²(SN) = 1454.9,  χ²(BAO) = 10.13,  тотал = 1465.0

Δχ² тотал (DSIC − ΛCDM) = −4.33   (ΔSN = −2.15,  ΔBAO = −2.18)
BAO χ²/dof (dof = 12−4 = 8):  DSIC = 0.99,  ΛCDM = 1.27
```

На совместных данных DSIC описывает геометрию расширения **не хуже, а формально чуть лучше** ΛCDM (Δχ² = −4.33 в пользу DSIC; по шкале Джеффриса это слабое, неубедительное предпочтение). Согласованность по BAO (`χ²/dof ≈ 1`) показывает, что та же геометрия `μ₀`, что описывает сверхновые, одновременно воспроизводит барионный масштаб.

> **Замечание о `r_d`.** При совместном фите `r_d ≈ 136 Мпк` ниже стандартных `≈ 147 Мпк`; это следствие высокой `H₀ ≈ 73` из SH0ES-якоря (произведение `r_d·H₀` фиксируется данными). При ранней шкале `H₀ ≈ 67.4` обратная задача из тех же `D_M/r_d`, `D_H/r_d` даёт `r_d ≈ 148 Мпк`, постоянное по `z`. Значение `r_d` смещается вместе с `H₀` через «напряжение Хаббла» одинаково в обеих моделях.

## 4. `Om(z)`-диагностика — честный результат

Диагностика `Om(z) = (E² − 1)/((1+z)³ − 1)`, `E = H/H₀`. У плоской ΛCDM `Om(z) = Ωm = const` (горизонталь); DSIC предсказывает U-образную `Om(z)` с минимумом около `z ≈ 2`. `Om(z)` строится **из данных** (хронометры `H(z)` Moresco + DESI `D_H/r_d`) и сравнивается с обеими кривыми.

### 4.1 Наивная нормировка (документ-предупреждение)

Если `H₀` извлечь, подгоняя хронометры ΛCDM-формой, данные оказываются ближе к DSIC:

```
χ²(DSIC U-кривая) = 17.9
χ²(ΛCDM плоская)  = 21.1
Δχ² = −3.3   (формально «ближе к DSIC»)
```

**Этот результат — артефакт нормировки** и не должен использоваться как свидетельство: способ извлечения `H₀` протаскивает модель в ответ.

### 4.2 Модельно-независимая нормировка (правильный тест)

Честная постановка: `H₀` берётся тремя независимыми способами, и проверяется, устойчив ли знак Δχ²:

| Нормировка `H₀` | `χ²_DSIC` | `χ²_ΛCDM` | Δχ² | Вердикт |
|---|---|---|---|---|
| Непарам. экстраполяция (`H₀ = 69.3`) | 26.8 | 23.1 | +3.7 | ближе ΛCDM |
| SH0ES (`H₀ = 73.0`) | 174.3 | 144.5 | +29.9 | ближе ΛCDM |
| Planck (`H₀ = 67.4`) | 21.0 | 28.5 | −7.5 | ближе DSIC |

Скан по `H₀` от 64 до 76:

```
H₀ = 64: Δχ² = −23.6      H₀ = 70: Δχ² =  +8.1
H₀ = 66: Δχ² = −14.7      H₀ = 72: Δχ² = +22.1
H₀ = 68: Δχ² =  −4.2      H₀ = 74: Δχ² = +38.1
H₀ = 69: Δχ² =  +1.7      H₀ = 76: Δχ² = +56.2
```

**Знак Δχ² меняется со знаком `H₀`** (диапазон −23.6 … +56.2, перемена около `H₀ ≈ 68.7`).

> **Вывод.** При текущих данных `Om(z)`-тест **не разрешает** DSIC и ΛCDM: результат полностью определяется выбором нормировки `H₀`, а не формой `Om(z)`. Заявлять победу DSIC по этой пробе нельзя. Тест станет решающим только при независимо закреплённой `H₀` и более точных `H(z)`.

## 5. Фальсифицируемые предсказания DSIC

Там, где данных пока мало, DSIC и ΛCDM расходятся настолько, что измерения вынесут вердикт. Параметры из совместного фита (`μ₀ = 0.7615`, `Ωm = 0.3044`).

### 5.1 Форма `Om(z)`

| `z` | `Om_DSIC` | `Om_ΛCDM` |
|---|---|---|
| 0.2 | 0.3576 | 0.3044 |
| 0.5 | 0.3307 | 0.3044 |
| 1.0 | 0.3075 | 0.3044 |
| 2.0 | 0.3046 | 0.3044 |
| 3.0 | 0.3283 | 0.3044 |
| 5.0 | 0.4066 | 0.3044 |

DSIC даёт U-образную `Om(z)` (минимум около `z ≈ 2`, подъём к высоким `z`) — качественная подпись, которой у плоской ΛCDM нет. Плоская `Om(z)`, измеренная точно, работала бы против DSIC.

### 5.2 Радиальный BAO на высоком `z` (Lyman-α)

| `z` | `D_H/r_d` DSIC | `D_H/r_d` ΛCDM | разница |
|---|---|---|---|
| 2.33 | 8.606 | 8.595 | +0.1 % |
| 2.5 | 7.972 | 8.009 | −0.5 % |
| 3.0 | 6.443 | 6.610 | −2.5 % |
| 3.5 | 5.294 | 5.569 | −4.9 % |

К `z ≈ 3.5` расхождение достигает ~5 %. Точные Lyman-α BAO в этом диапазоне — прямой тест.

### 5.3 Темп расширения за пределами данных

```
z = 5  : E_DSIC / E_ΛCDM = 1.15   (+15 %)
z = 10 : E_DSIC / E_ΛCDM = 1.47   (+47 %)
```

DSIC предсказывает существенно более жёсткую раннюю историю расширения. Любой зонд при `z > 3` (рост структур, `fσ8`, реликтовое излучение) способен опровергнуть модель. Это сильная сторона работы: DSIC делает резкие, проверяемые предсказания, а не подстраивается под наблюдения.

## 6. Итог

На геометрии расширения поздней Вселенной (`z < 2.3`) единственный параметр `μ₀ ≈ 0.76` согласованно описывает сверхновые и BAO, статистически неотличимо от ΛCDM и без тёмной энергии:

- сверхновые: Δχ² от −0.13 до +0.05 — неотличимо;
- устойчивость к способу учёта ошибок (STAT-ONLY): подтверждена;
- SN + BAO совместно: Δχ² = −4.33 (слабое формальное предпочтение DSIC);
- BAO `D_M`–`D_H`: `χ²/dof ≈ 0.99` против 1.27 у ΛCDM.

Что **не** подтверждено и заявлено честно:

- `Om(z)`-тест при текущих данных не разрешает модели — знак Δχ² зависит от нормировки `H₀`; ранее полученное «ближе к DSIC» — артефакт нормировки.

Где DSIC можно проверить:

- форма `Om(z)`, Lyman-α BAO при `z ≈ 3.5` (~5 % расхождение), `E(z)` при `z = 5–10` (15–47 % расхождение).

DSIC — жизнеспособная одно-параметрическая геометрическая альтернатива на поздних данных, с чёткими фальсифицируемыми предсказаниями на ранних. Текущие данные её не отвергают и не подтверждают сильнее ΛCDM; решающими станут высокий `z` и независимо закреплённая `H₀`.
